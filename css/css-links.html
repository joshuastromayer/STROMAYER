CSS Links

Links are an essential part of any website, allowing users to navigate between pages or sections. With CSS, you can style links to make them more visually appealing and interactive.

Basic Link Styling

The most common link element in HTML is the `<a>` tag. By default, links are styled with an underline and a specific color. You can customize these styles using CSS.

<style>
a {
  color: blue;
  text-decoration: none;
}
</style>

Interactive Link States

Links have different states that can be styled individually:

  • :link - Applies to unvisited links.
  • :visited - Applies to links the user has already visited.
  • :hover - Applies when the user hovers over the link.
  • :active - Applies to the link being clicked.
<style>
a:link {
  color: blue;
}
a:visited {
  color: purple;
}
a:hover {
  color: orange;
  text-decoration: underline;
}
a:active {
  color: red;
}
</style>
<a href="#">Interactive Link</a>

Removing Link Underlines

By default, links are underlined. To remove the underline, use the text-decoration property and set it to none.

<style>
a {
  text-decoration: none;
}
</style>
<a href="#">No Underline</a>

Hover Effects

Adding hover effects to links makes them more interactive and provides a better user experience. For example, you can change the text color or add an underline when the user hovers over a link.

<style>
a:hover {
  color: orange;
  text-decoration: underline;
}
</style>
<a href="#">Hover Me</a>

Styling Button-Like Links

You can style links to look like buttons by adding padding, background colors, and rounded corners.

<style>
a {
  display: inline-block;
  padding: 10px 20px;
  background-color: lightblue;
  color: black;
  text-decoration: none;
  border-radius: 5px;
}
a:hover {
  background-color: deepskyblue;
}
</style>
