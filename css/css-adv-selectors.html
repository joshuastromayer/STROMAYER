CSS Advanced Selectors

Advanced selectors in CSS allow you to target elements based on their relationship with other elements in the DOM. They provide powerful tools for styling complex layouts with precision.

Descendant Selector

The descendant selector targets all elements that are nested within a specified parent, regardless of how deeply nested they are. It is represented by a space between two selectors.

<style>
div p {
  color: blue;
}
</style>
<div>
  <p>This paragraph is inside a div.</p>
  <section>
    <p>This paragraph is inside a section inside a div.</p>
  </section>
</div>

Child Selector

The child selector (>) targets only the direct children of an element.

<style>
div > p {
  color: green;
}
</style>
<div>
  <p>This paragraph is a direct child of div.</p>
  <section>
    <p>This paragraph is not directly inside div.</p>
  </section>
</div>

Adjacent Sibling Selector

The adjacent sibling selector (+) targets an element that is immediately preceded by another element.

<style>
h1 + p {
  color: red;
}
</style>
<h1>Heading</h1>
<p>This paragraph follows the heading.</p>
<p>This paragraph is not targeted.</p>

General Sibling Selector

The general sibling selector (~) targets all siblings of a specified element that come after it.

<style>
h1 ~ p {
  color: purple;
}
</style>
<h1>Heading</h1>
<p>This paragraph is a sibling of the heading.</p>
<p>This is another sibling of the heading.</p>

CSS Advanced Selectors: Exercises

1. Descendant Selector

Create a structure with the following HTML:

  • A <div> element containing two <p> elements.
  • One of the <p> elements should be nested inside a <section> within the <div>.
Write a CSS rule to style all <p> elements inside the <div> with a blue color.

<style>
div p {
  color: blue;
}
</style>
<div>
  <p>Paragraph directly inside div.</p>
  <section>
    <p>Paragraph inside section within div.</p>
  </section>
</div>

2. Child Selector

Create a <div> containing the following:

  • One direct child <p> element.
  • A nested <section> that also contains a <p> element.
Write a CSS rule to style only the direct child <p> element with a green color.

<style>
div > p {
  color: green;
}
</style>
<div>
  <p>Direct child paragraph.</p>
  <section>
    <p>Nested paragraph.</p>
  </section>
</div>

3. Adjacent Sibling Selector

Create the following HTML structure:

  • A <h1> element.
  • A <p> element immediately after the <h1>.
  • A second <p> that does not immediately follow the <h1>.
Write a CSS rule to style only the first <p> (the one immediately after the <h1>) with a red color.

<style>
h1 + p {
  color: red;
}
</style>
<h1>Heading</h1>
<p>This paragraph is styled red.</p>
<p>This paragraph is not styled.</p>

4. General Sibling Selector

Create the following HTML structure:

  • A <h1> element.
  • Three <p> elements that follow the <h1>.
Write a CSS rule to style all <p> elements that are siblings of the <h1> with a purple color.

<style>
h1 ~ p {
  color: purple;
}
</style>
<h1>Heading</h1>
<p>Sibling 1 styled purple.</p>
<p>Sibling 2 styled purple.</p>
<p>Sibling 3 styled purple.</p>

5. Combining Selectors

Create a structure with:

  • A <div> containing multiple <p> elements.
  • An <h1> element outside the <div>.
Write CSS rules to:
  • Style all <p> elements inside the <div> with a blue color.
  • Style the <h1> and its first sibling <p> with red.

<style>
div p {
  color: blue;
}
h1, h1 + p {
  color: red;
}
</style>
<div>
  <p>Paragraph inside div styled blue.</p>
  <p>Another paragraph inside div styled blue.</p>
</div>
<h1>Heading styled red.</h1>
<p>First sibling of h1 styled red.</p>
<p>Other paragraph not styled red.</p>
