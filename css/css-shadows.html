CSS Shadows

CSS Shadows allow you to add depth and dimension to your designs by applying shadows to elements. You can use shadows on both text and boxes, with customizable properties like size, blur, color, and position.

Text Shadows

The text-shadow property adds a shadow to text. It takes the following values:

  • Horizontal Offset: Moves the shadow horizontally.
  • Vertical Offset: Moves the shadow vertically.
  • Blur Radius: Controls the blur of the shadow (optional).
  • Color: Specifies the color of the shadow.
<style>
h1 {
  text-shadow: 2px 2px 5px gray;
}
</style>
<h1>Text Shadow Example</h1>

In this example, the text shadow is offset by 2px horizontally and vertically, with a 5px blur radius and a gray color.

Box Shadows

The box-shadow property adds a shadow to boxes. It takes the following values:

  • Horizontal Offset: Moves the shadow horizontally.
  • Vertical Offset: Moves the shadow vertically.
  • Blur Radius: Controls the blur of the shadow (optional).
  • Spread Radius: Expands or contracts the size of the shadow (optional).
  • Color: Specifies the color of the shadow.
  • Inset: Makes the shadow appear inside the element instead of outside (optional).
<style>
div {
  width: 200px;
  height: 100px;
  background-color: lightblue;
  box-shadow: 4px 4px 10px gray;
}
</style>
<div></div>

This box shadow is offset by 4px horizontally and vertically, with a 10px blur radius and a gray color.

Adding Multiple Shadows

You can add multiple shadows to the same element by separating them with commas. For example:

<style>
div {
  width: 200px;
  height: 100px;
  background-color: lightgreen;
  box-shadow: 2px 2px 5px gray, -2px -2px 5px lightgray;
}
</style>
<div></div>

In this example, the element has two shadows: one offset positively and one offset negatively.

Inset Shadows

The inset keyword makes the shadow appear inside the element:

<style>
div {
  width: 200px;
  height: 100px;
  background-color: lightcoral;
  box-shadow: inset 3px 3px 5px black;
}
</style>
<div></div>

This creates a shadow that appears inside the box.

Practical Example

Combine box shadows and text shadows to create a visually appealing design:

<style>
h1 {
  text-shadow: 2px 2px 5px gray;
}
div {
  width: 250px;
  height: 150px;
  background-color: lightyellow;
  box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3);
}
</style>
<h1>Shadow Example</h1>

CSS Shadows: Exercises

1. Simple Text Shadow

Add a shadow to a heading (<h1>) with the following properties:

  • Horizontal offset: 3px
  • Vertical offset: 3px
  • Blur radius: 5px
  • Shadow color: gray

<style>
h1 {
  text-shadow: 3px 3px 5px gray;
}
</style>
<h1>Shadow Example</h1>

2. Simple Box Shadow

Create a square box with the following styles:

  • Width: 100px
  • Height: 100px
  • Background color: lightblue
Add a shadow with:
  • Horizontal offset: 4px
  • Vertical offset: 4px
  • Blur radius: 8px
  • Shadow color: black

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  box-shadow: 4px 4px 8px black;
}
</style>
<div></div>

3. Multiple Box Shadows

Create a rectangular box with the following styles:

  • Width: 150px
  • Height: 75px
  • Background color: lightgreen
Add two shadows:
  • The first shadow should have a 5px horizontal offset, 5px vertical offset, and 10px blur radius in gray.
  • The second shadow should have a -5px horizontal offset, -5px vertical offset, and 10px blur radius in lightgray.

<style>
div {
  width: 150px;
  height: 75px;
  background-color: lightgreen;
  box-shadow: 5px 5px 10px gray, -5px -5px 10px lightgray;
}
</style>
<div></div>

Create a box with the following styles:

  • Width: 150px
  • Height: 100px
  • Background color: lightyellow
Add an inset shadow with:
  • Horizontal offset: 3px
  • Vertical offset: 3px
  • Blur radius: 6px
  • Shadow color: darkgray

<style>
div {
  width: 150px;
  height: 100px;
  background-color: lightyellow;
  box-shadow: inset 3px 3px 6px darkgray;

<div></div>

5. Practical Shadow Combination

Create a button with the following styles:

  • Padding: 10px 20px
  • Background color: lightcoral
  • Border: none
  • Text color: white
Add:
  • A text shadow with 1px horizontal offset, 1px vertical offset, and 3px blur radius in black.
  • A box shadow with 2px horizontal offset, 2px vertical offset, and 5px blur radius in rgba(0, 0, 0, 0.5).

<style>
button {
  padding: 10px 20px;
  background-color: lightcoral;
  border: none;
  color: white;
  text-shadow: 1px 1px 3px black;
  box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
}
</style>
<button>Shadow Button</button>
