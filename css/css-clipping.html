CSS Clipping

CSS Clipping allows you to control the visibility of specific parts of an element by defining a clipping region. This can be achieved using the clip-path property, which provides a variety of ways to define complex shapes and boundaries for elements.

The clip-path Property

The clip-path property defines a visible area for an element. Anything outside this area is hidden. Common values include:

  • circle(): Defines a circular clipping area.
  • ellipse(): Defines an elliptical clipping area.
  • inset(): Defines a rectangular clipping area with optional rounded corners.
  • polygon(): Defines a custom polygonal clipping area.

Using clip-path with Shapes

1. Circle

Clip an element to a circular shape using the circle() function:

<style>
.circle {
  width: 200px;
  height: 200px;
  background-color: lightblue;
  clip-path: circle(50%);
}
</style>
<div class="circle"></div>

In this example, the element is clipped into a circular shape with a radius of 50% of its width and height.

2. Ellipse

Clip an element to an elliptical shape using the ellipse() function:

<style>
.ellipse {
  width: 300px;
  height: 200px;
  background-color: lightcoral;
  clip-path: ellipse(50% 30% at 50% 50%);
}
</style>
<div class="ellipse"></div>

This clips the element into an ellipse with the specified horizontal and vertical radii.

3. Inset

Clip an element to a rectangular shape using the inset() function:

<style>
.inset {
  width: 300px;
  height: 200px;
  background-color: lightgreen;
  clip-path: inset(20px 30px 20px 30px round 10px);
}
</style>
<div class="inset"></div>

This creates a rectangular clipping area with 20px padding on the top and bottom, 30px padding on the left and right, and rounded corners of 10px.

4. Polygon

Clip an element to a custom polygonal shape using the polygon() function:

<style>
.polygon {
  width: 300px;
  height: 200px;
  background-color: lightpink;
  clip-path: polygon(50% 0, 100% 100%, 0 100%);
}
</style>
<div class="polygon"></div>

This clips the element into a triangular shape by defining three points.

Using clip-path with Images

You can also apply clipping to images to create visually interesting effects:

<style>
.image {
  width: 300px;
  height: 300px;
  clip-path: circle(50%);
  overflow: hidden;
}
</style>
<img class="image" src="image.jpg" alt="Clipped Image">

This clips the image into a circular shape.

Practical Applications

The clip-path property is often used in UI design for:

  • Creating decorative shapes for elements or images.
  • Customizing masks for interactive design elements.
  • Highlighting specific areas in animations or transitions.
