CSS Padding

Padding is the space between the content of an element and its border. It controls the internal spacing within an element, providing breathing room for its content. Unlike margins, which create space outside an element, padding affects the inside.

How Padding Works

Padding is added inside an element’s border and around its content. It is invisible but creates space that separates the content from the element’s edges.

<style>
div {
  border: 1px solid black;
  padding: 20px;
}
</style>
<div>This div has 20px of padding.</div>

Specifying Padding

Padding can be specified using shorthand or individual properties for each side: padding-top, padding-right, padding-bottom, and padding-left.

<style>
div {
  border: 1px solid black;
  padding: 10px 15px 20px 25px; /* Top, Right, Bottom, Left */
}
</style>
<div>Custom padding for each side.</div>

Padding and Box Model

Padding is a key part of the CSS box model. It sits between the content and the border of an element. When combined with other box model properties (like margin and border), padding helps control the overall size of an element.

<style>
div {
  width: 200px;
  border: 2px solid black;
  padding: 20px;
}
</style>
<div>This box demonstrates padding.</div>

Padding vs. Margins

While padding adds space inside an element, margins create space outside an element. Both are used for spacing but serve different purposes. Here’s a visual comparison:

<style>
.with-padding {
  border: 1px solid black;
  padding: 20px;
  background-color: blue;
  color: white;
}
.with-margin {
  border: 1px solid black;
  margin: 20px;
  background-color: green;
  color: white;
}
</style>
<div class="with-padding">Padding Example</div>
<div class="with-margin">Margin Example</div>

Margin Example

Best Practices

- Use padding to create comfortable spacing for content inside an element.
- Avoid excessive padding as it might cause layout issues.
- Combine padding with other box model properties to create balanced designs.

CSS Padding: Exercises

1. Add Padding to a Paragraph

Create a paragraph with 20px padding on all sides. Add some text inside the paragraph.

<style>
p {
  padding: 20px;
  border: 1px solid black;
}
</style>
<p>This paragraph has padding of 20px on all sides.</p>

2. Add Different Padding

Create a div with different padding: 10px on the top, 15px on the right, 20px on the bottom, and 25px on the left.

<style>
div {
  padding: 10px 15px 20px 25px;
  border: 1px solid black;
}
</style>
<div>This div has different padding.</div>

3. Create a Box with Text Inside

Create a box (a div) with a width of 200px and height of 100px. Add 10px padding inside and some centered text.

<style>
div {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 1px solid black;
  text-align: center;
}
</style>
<div>This is a box with padding.</div>

4. Compare Padding and Margin

Create two divs: one with 20px padding and one with 20px margin. Add text to demonstrate the difference visually.

<style>
.padding-box {
  padding: 20px;
  border: 1px solid black;
}
.margin-box {
  margin: 20px;
  border: 1px solid black;
}
</style>
<div class="padding-box">This box has padding.</div>
<div class="margin-box">This box has margin.</div>

5. Combine Padding and Border

Create a div with a solid border and 15px padding inside. Add some text to see how the padding works with the border.

<style>
div {
  padding: 15px;
  border: 2px solid black;
}
</style>
<div>This div has padding and a border.</div>
