CSS Filters

CSS Filters allow you to apply visual effects to elements, such as blurring, brightness adjustment, color shifting, and more. These effects can be combined to create creative and visually appealing designs.

The filter Property

The filter property applies graphical effects to an element. It can accept one or multiple filter functions. The basic syntax is:

<style>
img {
  filter: blur(5px);
}
</style>
<img src="image.jpg" alt="Blurred Image">

In this example, the image is blurred with a radius of 5 pixels.

Common Filter Functions

Here are some commonly used filter functions:

  • blur(px): Applies a Gaussian blur to the element. The higher the value, the stronger the blur.
  • brightness(%): Adjusts the brightness of the element. Values above 100% increase brightness, and values below 100% decrease it.
  • contrast(%): Adjusts the contrast of the element. Values above 100% increase contrast, and values below 100% decrease it.
  • grayscale(%): Converts the element to grayscale. A value of 100% makes it fully grayscale, while 0% keeps the original color.
  • sepia(%): Converts the element to sepia tones. A value of 100% makes it fully sepia.
  • hue-rotate(deg): Rotates the hue of the element’s colors. The value is specified in degrees (e.g., 90deg).
  • invert(%): Inverts the colors of the element. A value of 100% fully inverts the colors.
  • saturate(%): Adjusts the color saturation. Values above 100% increase saturation, and values below 100% decrease it.
  • opacity(%): Adjusts the transparency of the element. A value of 100% is fully opaque, and 0% is fully transparent.

Examples of Filter Effects

1. Blur Effect

Apply a blur effect to an image:

<style>
.blur {
  filter: blur(10px);
}
</style>
<img class="blur" src="image.jpg" alt="Blurred Image">

2. Brightness Adjustment

Adjust the brightness of an image:

<style>
.bright {
  filter: brightness(150%);
}
</style>
<img class="bright" src="image.jpg" alt="Bright Image">

3. Grayscale Effect

Convert an image to grayscale:

<style>
.grayscale {
  filter: grayscale(100%);
}
</style>
<img class="grayscale" src="image.jpg" alt="Grayscale Image">

4. Multiple Filters

Combine multiple filters to create unique effects:

<style>
.combined {
  filter: brightness(120%) contrast(90%) sepia(50%);
}
</style>
<img class="combined" src="image.jpg" alt="Combined Filters Image">

Practical Applications

CSS Filters can be used for:

  • Creating hover effects (e.g., brighten an image when hovered).
  • Adjusting images for different themes (e.g., making them grayscale for a monochromatic theme).
  • Highlighting specific elements with color adjustments.

5. Hover Effect with Filters

Create a hover effect for an image where the following filters are applied on hover:

  • Brightness: 120%
  • Contrast: 110%
Ensure the filters are not applied when the image is not hovered.

<style>
.hover-filter {
  width: 300px;
  height: 200px;
  transition: filter 0.3s;
}
.hover-filter:hover {
  filter: brightness(120%) contrast(110%);
}
</style>
