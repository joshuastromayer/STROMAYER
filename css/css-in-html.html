CSS in HTML

CSS (Cascading Style Sheets) is used to style and visually enhance HTML content. While HTML structures the content of your webpage, CSS determines how it looks, including colors, fonts, spacing, layout, and more. To apply CSS to HTML, you need to link them together. This tutorial covers the various ways to use CSS with HTML.

Inline CSS

Inline CSS is applied directly within an HTML element using the style attribute. This method is useful for applying quick styles to specific elements, but it can make your code harder to maintain.

<p style="color: blue; font-size: 18px;">This is styled using inline CSS.</p>

Note: Inline CSS should be used sparingly to keep your HTML and CSS code organized.

Internal CSS

Internal CSS is written within a <style> tag inside the <head> section of your HTML document. This method is ideal for small projects or when styling a single webpage.

<!DOCTYPE html>
<html>
  <head>
    <style>
      p {
        color: green;
        font-weight: bold;
      }
    </style>
  </head>
  <body>
    <p>This is styled using internal CSS.</p>
  </body>
</html>

External CSS

External CSS is stored in a separate file with the extension .css. It is linked to your HTML file using the <link> tag in the <head> section. This method is the best practice for larger projects as it allows for easy reusability and better organization of styles.

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <p>This is styled using external CSS.</p>
  </body>
</html>

Choosing the Right Method

The choice of CSS method depends on the size and complexity of your project:

  • Inline CSS: Quick fixes or styling single elements.
  • Internal CSS: Small projects or single-page designs.
  • External CSS: Larger projects or when reusability and organization are key.

Best Practices

To ensure your code is clean and maintainable:

  • Use external CSS whenever possible.
  • Keep your styles organized and use meaningful class or ID names.
  • Avoid using inline CSS unless absolutely necessary.
