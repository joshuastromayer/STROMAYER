CSS Float and Clear

The float property in CSS allows elements to be positioned to the left or right of their container, while text or other inline elements wrap around them. The clear property is used to control the behavior of elements around floated elements.

Float Property

The float property is commonly used to create layouts or to wrap text around images. It can take the following values:

  • none: Default. The element does not float.
  • left: Floats the element to the left of its container.
  • right: Floats the element to the right of its container.
<style>
.float-left {
  float: left;
  width: 150px;
  margin: 10px;
  background-color: lightblue;
}
.float-right {
  float: right;
  width: 150px;
  margin: 10px;
  background-color: lightcoral;
}
</style>
<div class="float-left">This floats left.</div>
<div class="float-right">This floats right.</div>
<p>Content wraps around floated elements.</p>

Clear Property

The clear property is used to prevent elements from wrapping around floated elements. It can take the following values:

  • none: Default. Allows elements to wrap around floats.
  • left: Prevents wrapping around elements floated to the left.
  • right: Prevents wrapping around elements floated to the right.
  • both: Prevents wrapping around elements floated to either side.
<style>
.cleared {
  clear: both;
  margin-top: 10px;
  background-color: lightgreen;
}
</style>
<div class="float-left">Floated left.</div>
<div class="float-right">Floated right.</div>
<div class="cleared">Cleared both sides.</div>

Common Float Issues

Floating elements can sometimes cause layout issues. For example, a parent container may collapse if its child elements are floated. To fix this, you can use the "clearfix" technique.

<style>
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
.clearfix {
  background-color: lightgray;
  padding: 10px;
}
</style>
<div class="clearfix">
  <div class="float-left">Floated left.</div>
  <div class="float-right">Floated right.</div>
</div>

CSS Float and Clear: Exercises

1. Float Left

Create a <div> element with the following styles:

  • Width: 200px
  • Height: 100px
  • Background color: lightblue
  • Float: left
  • Margin: 10px

Observe how the content wraps around the floated element.

<style>
.float-left {
  width: 200px;
  height: 100px;
  background-color: lightblue;
  float: left;
  margin: 10px;
}
</style>
<div class="float-left">Floated Left</div>
<p>This text wraps around the floated element.</p>

2. Float Right

Create a <div> element with the following styles:

  • Width: 150px
  • Height: 80px
  • Background color: lightcoral
  • Float: right
  • Margin: 10px

Observe how the content wraps around the floated element on the opposite side.

<style>
.float-right {
  width: 150px;
  height: 80px;

<div class="float-right">Floated Right</div>

3. Clear Both

Create two floated <div> elements (one floated left and the other floated right) followed by a <div> with clear: both. Style them with:

  • Floated elements: Width 100px, Height 50px, Background color lightgreen, Margin 5px
  • Cleared element: Width 100%, Background color lightgray

Observe how the cleared element starts below the floated elements.

<style>
.float-left {
  width: 100px;
  height: 50px;
  background-color: lightgreen;
  float: left;
  margin: 5px;
}
.float-right {
  width: 100px;
  height: 50px;
  background-color: lightgreen;
  float: right;
  margin: 5px;
}
.cleared {
  clear: both;
  width: 100%;
  background-color: lightgray;
}
</style>
<div class="float-left">Left</div>
<div class="float-right">Right</div>
<div class="cleared">Cleared Element</div>

4. Clearfix

Create a container <div> with two floated child <div> elements. Use the clearfix technique to ensure the container fully wraps around its floated children. Style:

  • Container: Background color lightgray
  • Child elements: Width 120px, Height 60px, Background color lightpink, Float left
<style>
.container::after {
  content: "";
  display: block;
  clear: both;
}
.container {
  background-color: lightgray;
  padding: 10px;
}
.child {
  width: 120px;
  height: 60px;
  background-color: lightpink;
  float: left;
  margin: 5px;
}
</style>
<div class="container">
  <div class="child">Child 1</div>
  <div class="child">Child 2</div>
</div>
