CSS Responsive Design

Responsive design ensures that a website looks and functions well across a wide range of devices, screen sizes, and orientations. It adapts the layout, styles, and content to provide a consistent user experience.

Core Principles of Responsive Design

To create responsive designs, consider the following principles:

  • Fluid Grids: Use percentages or fractional units for widths instead of fixed pixel values.
  • Flexible Images: Ensure images resize within their containers without overflowing.
  • Media Queries: Adjust styles based on the user's device or viewport size.

Using Fluid Grids

A fluid grid divides the page layout into flexible columns. Instead of using fixed widths (e.g., 200px), use percentages or the fr unit:

<style>
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}
.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
</style>
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>

Flexible Images

To make images responsive, use the max-width property. Setting max-width: 100% ensures that images resize to fit their containers:

<style>
img {
  max-width: 100%;
  height: auto;
}
</style>
<img src="example.jpg" alt="Responsive Image">

Combining Media Queries

Media queries allow you to adapt layouts based on breakpoints. For example:

<style>
.container {
  display: flex;
  flex-wrap: wrap;
}
.box {
  width: 30%;
  margin: 1%;
  background-color: lightgreen;
  text-align: center;
  padding: 10px;
}
@media (max-width: 768px) {
  .box {
    width: 100%;
  }
}
</style>
<div class="container">
  <div class="box">Box 1</div>
  <div class="box">Box 2</div>
  <div class="box">Box 3</div>
</div>

In this example, the boxes are arranged in a row for larger screens. On smaller screens (768px or smaller), they stack vertically.

Viewport Meta Tag

The <meta> viewport tag is essential for responsive design. It ensures that the page scales correctly on different devices. Add this to the <head> of your HTML:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Practical Example

Combine these techniques to create a responsive layout. For example:

<style>
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
.box {
  background-color: lightcoral;
  padding: 20px;
  text-align: center;
}
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
</style>
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
</div>

CSS Responsive Design: Exercises

1. Fluid Grid

Create a responsive grid layout using the following specifications:

  • Three columns, each taking 1/3 of the available width.
  • A gap of 15px between columns.
  • When the viewport width is 768px or smaller, switch to a single-column layout.

<style>
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
}
.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
</style>
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>

2. Flexible Images

Add an image to your page and make it responsive by ensuring it never overflows its container. Use the following specifications:

  • Set the image to resize proportionally.
  • Ensure the container width is 80% of the viewport width.

<style>
.container {
  width: 80%;
  margin: auto;
}
img {
  max-width: 100%;
  height: auto;
}
</style>
<div class="container">
  <img src="example.jpg" alt="Responsive Image">
</div>

3. Hiding Elements

Create a navigation bar (<nav>) and hide it when the viewport width is smaller than 600px. Use display: none to hide the navigation bar.

<style>
nav {
  background-color: lightgreen;
  padding: 10px;
}
@media (max-width: 600px) {
  nav {
    display: none;
  }
}
</style>
<nav>Navigation Bar</nav>

4. Adjusting Font Sizes

Use media queries to adjust the font size of your page's content based on the viewport width:

  • For viewports 600px or smaller, set the font size to 14px.
  • For viewports between 601px and 1200px, set the font size to 16px.
  • For viewports larger than 1200px, set the font size to 18px.

<style>
body {
  font-size: 16px;
}
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
@media (min-width: 1201px) {
  body {
    font-size: 18px;
  }
}
</style>

5. Responsive Layout with Flexbox

Create a responsive layout using Flexbox with the following specifications:

  • Three boxes aligned horizontally on larger screens.
  • On screens smaller than 768px, stack the boxes vertically.

<style>
.container {
  display: flex;
  gap: 10px;
}
.box {
  background-color: lightcoral;
  padding: 20px;
  text-align: center;
}
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
</style>
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
