CSS Media Queries

Media queries in CSS allow you to apply styles based on the characteristics of the user's device or viewport, such as its width, height, orientation, or resolution. This makes it a powerful tool for creating responsive designs that adapt to different screen sizes and devices.

Basic Syntax

A media query consists of a @media rule followed by a condition and a block of CSS rules to apply if the condition is true. For example:

<style>
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
</style>

In this example, the background color of the body changes to lightblue if the viewport width is 600px or less.

Using Multiple Conditions

You can use logical operators like and, or, and not to combine or negate conditions in media queries:

  • and: Combine multiple conditions.
  • or (comma-separated queries): Apply if any condition is true.
  • not: Negate a condition.
<style>
@media (min-width: 601px) and (max-width: 1200px) {
  body {
    background-color: lightgreen;
  }
}
@media not (min-width: 601px) {
  body {
    background-color: lightcoral;
  }
}
</style>

In these examples, the first rule applies when the viewport width is between 601px and 1200px. The second rule applies when the viewport width is less than 601px.

Common Use Cases

Media queries are commonly used to create responsive designs. Here are some typical use cases:

  • Changing font sizes or styles based on screen size.
  • Adjusting the layout for mobile, tablet, and desktop devices.
  • Hiding or showing elements based on device characteristics.
<style>
@media (max-width: 768px) {
  nav {
    display: none;
  }
}
@media (orientation: landscape) {
  body {
    font-size: 18px;
  }
}
</style>

The first rule hides the navigation bar on screens 768px or smaller. The second rule increases the font size when the device is in landscape orientation.

Breakpoints

Breakpoints are specific widths where your design adapts to different devices. Common breakpoints include:

  • 320px–480px: Small devices (e.g., phones).
  • 481px–768px: Medium devices (e.g., tablets).
  • 769px–1024px: Large devices (e.g., small laptops).
  • 1025px–1200px: Extra-large devices (e.g., desktops).

For example:

<style>
@media (max-width: 480px) {
  body {
    background-color: lightblue;
  }
}
@media (min-width: 481px) and (max-width: 768px) {
  body {
    background-color: lightgreen;
  }
}
@media (min-width: 769px) {
  body {
    background-color: lightcoral;
  }
}
</style>

CSS Media Queries: Exercises

1. Basic Media Query

Create a media query that changes the background color of the page to lightblue when the viewport width is 600px or smaller. Write the CSS rule and test it by resizing your browser window.

<style>
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
</style>

2. Multiple Conditions

Write a media query that applies the following styles only when the viewport width is between 601px and 1200px:

  • Change the font size of all <p> elements to 18px.
  • Set the background color of the body to lightgreen.

<style>
@media (min-width: 601px) and (max-width: 1200px) {
  p {
    font-size: 18px;
  }
  body {
    background-color: lightgreen;
  }
}
</style>

3. Orientation-Based Styles

Create a media query that applies the following styles when the device is in landscape orientation:

  • Set the text color of all headings (<h1>–<h6>) to darkblue.
  • Change the background color of the body to lightyellow.

<style>
@media (orientation: landscape) {
  h1, h2, h3, h4, h5, h6 {
    color: darkblue;
  }
  body {
    background-color: lightyellow;
  }
}
</style>

4. Hiding Elements

Create a media query that hides the navigation bar (a <nav> element) when the viewport width is smaller than 768px. Use the display: none property to hide the element.

<style>
@media (max-width: 768px) {
  nav {
    display: none;
  }
}
</style>

5. Breakpoints for Layout

Use media queries to implement the following layout adjustments:

  • For viewports smaller than 600px, set the font size of the body to 14px.
  • For viewports between 600px and 1200px, set the font size of the body to 16px.
  • For viewports larger than 1200px, set the font size of the body to 18px.
Write the CSS code for these breakpoints.

<style>
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
@media (min-width: 601px) and (max-width: 1200px) {
  body {
    font-size: 16px;
  }
}
@media (min-width: 1201px) {
  body {
    font-size: 18px;
  }
}
</style>

6. Complex Media Query

Write a media query that applies the following styles only when the viewport width is between 768px and 1024px, and the device is in portrait orientation:

  • Set the background color of the body to lightgray.
  • Set the font size of all paragraphs (<p>) to 20px.

<style>
@media (min-width: 768px) and (max-width: 1024px) and (orientation: portrait) {
  body {
    background-color: lightgray;
  }
  p {
    font-size: 20px;
  }
}
</style>
