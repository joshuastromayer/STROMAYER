CSS Animations

CSS Animations allow you to create complex animations by defining keyframes that specify the styles at various points in the animation. They provide more control and flexibility than transitions, enabling you to animate multiple properties and create continuous effects.

Basic Syntax

A CSS Animation is defined using the @keyframes rule and applied to an element using the animation property. The @keyframes rule specifies the animation's behavior at different points, called keyframes.

<style>
@keyframes example {
  from { background-color: lightblue; }
  to { background-color: lightcoral; }
}
div {
  width: 200px;
  height: 100px;
  background-color: lightblue;
  animation: example 2s ease-in-out infinite;
}
</style>
<div></div>

In this example, the background color transitions between lightblue and lightcoral over 2 seconds, repeating infinitely.

Defining Keyframes

Keyframes define the intermediate steps of an animation. You can use percentages to specify when each keyframe occurs:

<style>
@keyframes slide {
  0% { transform: translateX(0); }
  50% { transform: translateX(100px); }
  100% { transform: translateX(0); }
}
div {
  width: 100px;
  height: 100px;
  background-color: lightgreen;
  animation: slide 3s ease infinite;
}
</style>
<div></div>

In this example, the element moves 100px to the right at 50% of the animation and returns to its starting position at 100%.

Animation Properties

The animation shorthand property combines several individual animation properties:

  • animation-name: Specifies the name of the @keyframes animation.
  • animation-duration: Specifies how long the animation takes (e.g., 2s).
  • animation-timing-function: Specifies the speed curve (e.g., ease, linear).
  • animation-delay: Specifies a delay before the animation starts.
  • animation-iteration-count: Specifies the number of times the animation repeats (infinite for continuous looping).
  • animation-direction: Specifies whether the animation alternates direction (normal, reverse, alternate).
<style>
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-50px); }
}
div {
  width: 100px;
  height: 100px;
  background-color: lightpink;
  animation: bounce 1s ease infinite alternate;
}
</style>
<div></div>

Controlling Animations

You can pause, play, or restart animations using the animation-play-state property:

<style>
div {
  width: 100px;
  height: 100px;
  background-color: lightyellow;
  animation: example 2s infinite;
  animation-play-state: paused;
}
div:hover {
  animation-play-state: running;
}
</style>
<div></div>

In this example, the animation starts only when you hover over the element.

Practical Example

Combine multiple properties to create a more complex animation:

<style>
@keyframes grow-shrink {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.5); }
}
div {
  width: 150px;
  height: 150px;
  background-color: lightblue;
  animation: grow-shrink 2s ease-in-out infinite;
}
</style>
<div></div>

CSS Animations: Exercises

1. Simple Animation

Create a box that changes its background color from blue to green over 3 seconds. The animation should repeat infinitely.

<style>
@keyframes color-change {
  from { background-color: blue; }
  to { background-color: green; }
}
div {
  width: 150px;
  height: 150px;
  background-color: blue;
  animation: color-change 3s infinite;
}
</style>
<div></div>

2. Keyframes with Percentages

Create a box that:

  • Starts with a width of 100px.
  • Expands to 200px at 50% of the animation.
  • Returns to 100px at the end of the animation.
The animation should last 2 seconds and loop infinitely.

<style>
@keyframes expand {
  0% { width: 100px; }
  50% { width: 200px; }
  100% { width: 100px; }
}
div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  animation: expand 2s infinite;
}
</style>
<div></div>

3. Timing Functions

Create a circle that grows to 1.5 times its original size over 1 second. Use the ease-in-out timing function and make the animation alternate directions.

<style>
@keyframes grow {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.5); }
}
div {
  width: 100px;
  height: 100px;
  background-color: lightpink;
  border-radius: 50%;
  animation: grow 1s ease-in-out infinite alternate;
}
</style>
<div></div>

4. Delay and Iteration Count

Create a square that rotates 45 degrees. The animation should:

  • Start after a 1-second delay.
  • Last for 1.5 seconds.
  • Repeat exactly 3 times.

<style>
@keyframes rotate {
  from { transform: rotate(0); }
  to { transform: rotate(45deg); }
}
div {
  width: 100px;
  height: 100px;
  background-color: lightgreen;
  animation: rotate 1.5s 1s 3;
}
</style>
<div></div>

5. Animation Direction

Create a rectangle that alternates between moving left and right. Use animation-direction: alternate to achieve this effect, and set the duration to 2 seconds.

<style>
@keyframes slide {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}
div {
  width: 150px;
  height: 75px;
  background-color: lightyellow;
  animation: slide 2s ease infinite alternate;
}
</style>
