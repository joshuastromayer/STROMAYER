CSS Variables

CSS Variables, also known as custom properties, are reusable values defined in your CSS that can be applied throughout your stylesheet. They make it easier to manage and maintain consistent styling, especially for themes, colors, or repeated values.

Defining and Using Variables

CSS Variables are defined using the -- prefix and are applied using the var() function. Variables are typically declared inside a :root selector for global scope:

<style>
:root {
  --main-bg-color: lightblue;
  --main-text-color: darkblue;
}
body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}
</style>
<body>This is a CSS Variables example.</body>

In this example, the background color and text color of the body element are set using CSS Variables.

Variable Scope

CSS Variables are scoped, meaning they can be declared within specific selectors and applied only within those selectors:

<style>
.card {
  --card-bg-color: lightyellow;
  --card-border-color: darkgoldenrod;
}
.card {
  background-color: var(--card-bg-color);
  border: 2px solid var(--card-border-color);
  padding: 20px;
}
</style>
<div class="card">This is a card.</div>

In this example, the variables --card-bg-color and --card-border-color are applied only within elements with the card class.

Fallback Values

The var() function allows you to specify a fallback value in case the variable is not defined:

<style>
body {
  background-color: var(--undefined-bg-color, lightgray);
}
</style>
<body>This uses a fallback value.</body>

If --undefined-bg-color is not defined, the background color will default to lightgray.

Updating Variables Dynamically

You can update CSS Variables dynamically using JavaScript to create interactive designs:

<style>
:root {
  --main-bg-color: lightblue;
}
body {
  background-color: var(--main-bg-color);
}
</style>
<button onclick="changeColor()">Change Color</button>
<script>
function changeColor() {
  document.documentElement.style.setProperty('--main-bg-color', 'lightcoral');
}
</script>

Clicking the button changes the background color dynamically by updating the --main-bg-color variable.

Advantages of CSS Variables

CSS Variables offer several advantages:

  • Reusability: Define once and use across multiple elements.
  • Maintainability: Easily update styles by changing variable values.
  • Dynamic Updates: Change values dynamically with JavaScript.
  • Customizable Themes: Quickly implement and switch between design themes.

CSS Variables: Exercises

1. Basic Variable Usage

Define two CSS variables in the :root selector:

  • --primary-color: lightblue
  • --text-color: darkblue
Apply these variables to style a <div> with:
  • Background color: --primary-color
  • Text color: --text-color

<style>
:root {
  --primary-color: lightblue;
  --text-color: darkblue;
}
div {
  background-color: var(--primary-color);
  color: var(--text-color);
  padding: 20px;
  text-align: center;
}
</style>
<div>This is a CSS Variables example.</div>

2. Scoped Variables

Create a card with the following styles:

  • Define two scoped variables in the .card class:
    • --card-bg: lightyellow
    • --card-border: darkgoldenrod
  • Use these variables for the card's background color and border color.

<style>
.card {
  --card-bg: lightyellow;
  --card-border: darkgoldenrod;
  background-color: var(--card-bg);
  border: 2px solid var(--card-border);
  padding: 20px;
  width: 200px;
  text-align: center;
}
</style>
<div class="card">This is a card.</div>

3. Fallback Values

Use the var() function with a fallback value to style a box. Define:

  • Background color: A CSS variable named --box-bg.
  • Fallback color: lightgray (if --box-bg is not defined).

<style>
div {
  background-color: var(--box-bg, lightgray);
  width: 100px;
  height: 100px;
  text-align: center;
  line-height: 100px;
}
</style>
<div>Fallback Example</div>

4. Dynamic Updates with JavaScript

Create a button that dynamically changes the background color of the page to lightcoral when clicked. Use the following steps:

  • Define a CSS variable --bg-color in the :root selector with an initial value of lightblue.
  • Use JavaScript to update --bg-color to lightcoral when the button is clicked.

<style>
:root {
  --bg-color: lightblue;
}
body {
  background-color: var(--bg-color);
  margin: 0;
  padding: 20px;
}
button {
  padding: 10px 20px;
  background-color: darkblue;
  color: white;
  border: none;
  cursor: pointer;
}
</style>
<button onclick="changeColor()">Change Background</button>
<script>
function changeColor() {
  document.documentElement.style.setProperty('--bg-color', 'lightcoral');
}
</script>

5. Practical Example: Themed Cards

Create two cards with the same structure but different themes. Define:

  • Variables for each theme (e.g., light and dark) inside their respective classes.
  • Apply these variables to style the cards differently.

<style>
.light-theme {
  --card-bg: white;
  --card-text: black;
  --card-border: gray;
}
.dark-theme {
  --card-bg: black;
  --card-text: white;
  --card-border: lightgray;
}
.card {
  background-color: var(--card-bg);
  color: var(--card-text);
  border: 1px solid var(--card-border);
  padding: 20px;
  width: 200px;
  text-align: center;
  margin-bottom: 20px;
}
</style>
<div class="card light-theme">Light Theme</div>
<div class="card dark-theme">Dark Theme</div>
