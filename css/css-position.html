CSS Position

The position property in CSS is used to control the placement of an element on a webpage. It defines how an element is positioned in the document and interacts with other elements. There are five main values for the position property: static, relative, absolute, fixed, and sticky.

Static Positioning

This is the default positioning method. Elements are positioned according to the normal document flow, and the top, right, bottom, and left properties do not apply.

<style>
div {
  border: 1px solid black;
  padding: 10px;
  margin: 10px;
}
</style>
<div>This is a static element.</div>

Relative Positioning

An element positioned relatively is moved relative to its normal position in the document flow. You can use the top, right, bottom, and left properties to adjust its position.

<style>
div {
  position: relative;
  top: 20px;
  left: 15px;
  border: 1px solid black;
  padding: 10px;
}
</style>
<div>This is a relatively positioned element.</div>

Absolute Positioning

An element positioned absolutely is removed from the normal document flow and positioned relative to the nearest positioned ancestor (an element with position set to relative, absolute, or sticky). If no positioned ancestor exists, it is positioned relative to the <html> element.

<style>
.container {
  position: relative;
  border: 1px solid black;
  padding: 10px;
  height: 100px;
}
.absolute-box {
  position: absolute;
  top: 10px;
  right: 10px;
  border: 1px solid red;
  padding: 5px;
}
</style>
<div class="container">
  Container Box
  <div class="absolute-box">Absolute Box</div>
</div>

Fixed Positioning

An element positioned with fixed is removed from the document flow and positioned relative to the viewport. It remains in the same position even when the page is scrolled.

<style>
div {
  position: fixed;
  top: 0;
  right: 0;
  border: 1px solid blue;
  padding: 10px;
}
</style>
<div>This is a fixed element.</div>

Sticky Positioning

A sticky element toggles between relative and fixed positioning depending on the scroll position. It behaves like a relatively positioned element until it crosses a specified threshold, after which it behaves like a fixed element.

<style>
div {
  position: sticky;
  top: 50px;
  border: 1px solid green;
  padding: 10px;
}
</style>
<div>This is a sticky element.</div>

CSS Position: Exercises

1. Static Position

Create a `<div>` with a static position and add the following styles:

  • Width: 200px
  • Height: 100px
  • Background color: lightblue

Observe how it follows the normal document flow.

<style>
div {
  width: 200px;
  height: 100px;
  background-color: lightblue;
}
</style>
<div>Static Box</div>

2. Relative Position

Create a `<div>` with a relative position and add the following styles:

  • Width: 250px
  • Height: 120px
  • Background color: lightcoral
  • Move it 20px down and 15px to the right using the top and left properties.
<style>
div {
  width: 250px;
  height: 120px;
  background-color: lightcoral;
  position: relative;
  top: 20px;
  left: 15px;
}
</style>
<div>Relative Box</div>

3. Absolute Position

Create a parent `<div>` with relative positioning and an inner `<div>` with absolute positioning. Add the following styles:

  • Parent width: 300px, height: 150px, background color: lightgray
  • Child width: 100px, height: 50px, background color: lightgreen
  • Position the child at the bottom-right corner of the parent.
<style>
.parent {
  position: relative;
  width: 300px;
  height: 150px;
  background-color: lightgray;
}
.child {
  position: absolute;
  width: 100px;
  height: 50px;
  background-color: lightgreen;
  bottom: 0;
  right: 0;
}
</style>
<div class="parent">
  <div class="child">Absolute Box</div>
</div>

4. Fixed Position

Create a `<div>` with a fixed position and add the following styles:

  • Width: 200px
  • Height: 50px
  • Background color: lightyellow
  • Position it at the top-right corner of the viewport.
<style>
div {
  position: fixed;
  width: 200px;
  height: 50px;
  background-color: lightyellow;
  top: 0;
  right: 0;
}
</style>
<div>Fixed Box</div>

5. Sticky Position

Create a sticky header and add the following styles:

  • Width: 100%
  • Height: 60px
  • Background color: lightpink
  • The header should stick to the top of the page when scrolling down.
<style>
header {
  position: sticky;
  top: 0;
  width: 100%;
  height: 60px;
  background-color: lightpink;
}
</style>
<header>Sticky Header</header>
