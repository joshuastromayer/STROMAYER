CSS Lists

Lists in HTML are a great way to organize content, and CSS allows you to style them in various ways to make them visually appealing and fit your design.

Types of Lists

HTML offers three main types of lists:

  • Ordered Lists (<ol>): Displays items with numbers or letters.
  • Unordered Lists (<ul>): Displays items with bullets or other markers.
  • Definition Lists (<dl>): Displays terms and their descriptions.

Styling List Markers

You can use the list-style-type property to change the markers of lists:

<style>
ul {
  list-style-type: square;
}
ol {
  list-style-type: upper-roman;
}
</style>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Removing List Markers

To remove the default markers, use list-style-type: none;:

<style>
ul {
  list-style-type: none;
}
</style>
<ul>
  <li>No markers here!</li>
  <li>Just plain text</li>
</ul>

Custom List Markers

You can use images or custom markers with the list-style-image property:

<style>
ul {
  list-style-image: url('marker.png');
}
</style>
<ul>
  <li>Custom marker 1</li>
  <li>Custom marker 2</li>
</ul>

Nested Lists

Lists can be nested inside each other, and you can style each level differently using CSS selectors:

<style>
ul ul {
  list-style-type: circle;
}
ul ul ul {
  list-style-type: square;
}
</style>
<ul>
  <li>Level 1</li>
  <ul>
    <li>Level 2</li>
    <ul>
      <li>Level 3</li>
    </ul>
  </ul>
</ul>

Controlling List Position

The list-style-position property controls whether the marker is inside or outside the list item:

<style>
ul {
  list-style-position: inside;
}
</style>
<ul>
  <li>Marker is inside the text.</li>
</ul>

CSS Lists: Exercises

1. Change List Markers

Create an unordered list (`<ul>`) with three items. Change the list markers to squares using CSS.

<style>
ul {
  list-style-type: square;
}
</style>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

2. Remove List Markers

Create an ordered list (`<ol>`) with three items. Remove the default list markers using CSS.

<style>
ol {
  list-style-type: none;
}
</style>
<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>

3. Use Custom Markers

Create an unordered list (`<ul>`) with three items. Use a custom image as the list marker.

<style>
ul {
  list-style-image: url('marker.png');
}
</style>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

4. Style Nested Lists

Create a nested list (`<ul>` inside another `<ul>`). Use circles as the marker for the inner list and squares for the outer list.

<style>
ul {
  list-style-type: square;
}
ul ul {
  list-style-type: circle;
}
</style>
<ul>
  <li>Outer Item 1</li>
  <li>Outer Item 2
    <ul>
      <li>Inner Item 1</li>
      <li>Inner Item 2</li>
    </ul>
  </li>
</ul>

5. List Markers Inside and Outside

Create an unordered list with three items. Place the list markers inside the text for one list and outside for another.

<style>
ul.inside {
  list-style-position: inside;
}
ul.outside {
  list-style-position: outside;
}
</style>
<ul class="inside">
  <li>Inside marker 1</li>
  <li>Inside marker 2</li>
</ul>
<ul class="outside">
  <li>Outside marker 1</li>
  <li>Outside marker 2</li>
</ul>
