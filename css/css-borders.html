CSS Borders

Borders in CSS are used to surround an element, helping to define its edges and improve the visual structure of your webpage. You can customize borders in terms of width, style, and color to create visually appealing designs.

Basic Border Properties

The following are the essential properties for working with borders:

  • border-width: Specifies the thickness of the border.
  • border-style: Specifies the style of the border (e.g., solid, dashed, dotted).
  • border-color: Specifies the color of the border.
<style>
div {
  border-width: 2px;
  border-style: solid;
  border-color: blue;
}
</style>
<div>This is a bordered div</div>

Shorthand Border Property

CSS provides a shorthand property, border, to specify all three properties (width, style, and color) in a single line.

<style>
div {
  border: 2px dashed green;
}
</style>
<div>This div has a dashed border</div>

Border Radius

The border-radius property allows you to create rounded corners for an element. You can specify the radius in pixels or percentages.

<style>
div {
  border: 2px solid red;
  border-radius: 10px;
}
</style>
<div>This div has rounded corners</div>

Individual Border Sides

You can apply different border styles, widths, and colors to each side of an element using the following properties:

  • border-top
  • border-right
  • border-bottom
  • border-left
<style>
div {
  border-top: 4px solid black;
  border-right: 2px dotted blue;
  border-bottom: 3px dashed red;
  border-left: 5px double green;
}
</style>
<div>This div has custom borders on each side</div>

Transparent and Hidden Borders

The border-style property allows you to set borders to transparent (completely invisible but still takes up space) or hidden (invisible and does not take up space).

<style>
div {
  border: 3px solid transparent;
}
</style>
<div>This div has a transparent border</div>

Practical Example

Here’s an example combining different border properties:

<style>
div {
  width: 200px;
  height: 100px;
  border: 3px solid black;
  border-radius: 15px;
  background-color: lightyellow;
}
</style>
<div>Custom border example</div>
