HTML Divs

A <div> (short for division) is one of the most versatile and commonly used elements in HTML. It is a container element used to group content together, which can then be styled or manipulated as a single unit.

What is a Div?

A <div> is essentially a block-level element that is used to organize content. It doesn’t have any inherent visual representation on its own—it’s purely structural. However, with CSS, you can style it to create visually appealing and organized layouts.

<div>
  <p>This is a paragraph inside a div.</p>
  <p>This is another paragraph inside the same div.</p>
</div>

Grouping Content

The main purpose of a <div> is to group elements together. This is especially useful when you want to apply the same styling or behavior to multiple elements.

<div>
  <h2>Section Title</h2>
  <p>This is a paragraph in the section.</p>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</div>

Divs and Layout

While divs themselves don’t create layouts, they are essential building blocks for layouts when combined with CSS. By using CSS, you can customize divs to control alignment, spacing, and more.

Examples of what you can achieve with CSS and divs include horizontal or vertical alignment of content, flexible grids, and responsive designs. While this topic focuses only on HTML, keep in mind that divs are a powerful tool when styled with CSS.

Nested Divs

Divs can be nested inside each other to create complex structures. Nested divs are helpful for creating hierarchical layouts where each section can be independently styled or organized.

<div>
  <div>
    <p>This is inside a nested div.</p>
  </div>
  <p>This is inside the parent div but outside the nested div.</p>
</div>

Best Practices for Using Divs

Although divs are incredibly versatile, it's important to use them responsibly to maintain semantic HTML:

  • Use divs for grouping and layout purposes.
  • For specific types of content (e.g., navigation, headers, footers), use semantic elements like <header>, <nav>, or <footer> instead of a <div>.
  • Give your divs meaningful class or ID names for easier identification and styling.
<div class="content-section">
  <h2>Welcome</h2>
  <p>This is a content section.</p>
</div>

Example: Simple Page Layout

Here’s an example of how divs can be used to structure a simple webpage layout:

<div class="header">
  <h1>My Website</h1>
</div>
<div class="main-content">
  <p>Welcome to my website!</p>
</div>
<div class="footer">
  <p>Copyright 2025</p>
</div>

HTML Divs: Exercises

1. Group Paragraphs Inside a Div

Create a div that contains two paragraphs with the following text:

  • First paragraph: "HTML Divs are versatile!"
  • Second paragraph: "They help structure your content."

<div>
  <p>HTML Divs are versatile!</p>
  <p>They help structure your content.</p>
</div>

2. Create a Section with a Header and List

Use a div to group the following elements:

  • An h2 heading with the text: "My Favorite Fruits"
  • An unordered list with items: "Apples", "Bananas", "Cherries"

<div>
  <h2>My Favorite Fruits</h2>
  <ul>
    <li>Apples</li>
    <li>Bananas</li>
    <li>Cherries</li>
  </ul>
</div>

3. Use Nested Divs

Create a parent div with the class "container" and a nested div inside it with the class "content". The nested div should contain the text: "Nested divs are powerful!".

<div class="container">
  <div class="content">
    Nested divs are powerful!
  </div>
</div>

4. Simple Page Layout

Create a basic page layout using divs with the following structure:

  • A div with the class "header" containing an h1 with the text: "Welcome to My Page"
  • A div with the class "main" containing a paragraph with the text: "This is the main content."
  • A div with the class "footer" containing a paragraph with the text: "Footer Information"

<div class="header">
  <h1>Welcome to My Page</h1>
</div>
<div class="main">
  <p>This is the main content.</p>
</div>
<div class="footer">
  <p>Footer Information</p>
</div>

5. Add a Class and ID

Create a div with the class "highlighted" and the id "unique-section". Inside it, add a paragraph with the text: "This is a unique section!".

<div class="highlighted" id="unique-section">
  <p>This is a unique section!</p>
</div>
