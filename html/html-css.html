Introduction to CSS

CSS (Cascading Style Sheets) is a powerful tool used to style and layout web pages. While HTML is used to structure content, CSS defines how that content looks, from colors and fonts to spacing and positioning.

What is CSS?

CSS is a language that describes the style of an HTML document. It controls the appearance of elements on a web page, making it possible to create visually stunning websites and interfaces.

<style>
  body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
  }
  h1 {
    color: navy;
    text-align: center;
  }
</style>

In this example, CSS is used to set the background color of the page, define the font for all text, and style the heading.

Where to Write CSS

  • Inline CSS: Written directly inside an HTML element’s style attribute.
  • Internal CSS: Placed within a <style> tag inside the HTML document’s <head>.
  • External CSS: Written in a separate file with a .css extension and linked to the HTML document using a <link> tag.
<!-- External CSS Example -->
<link rel="stylesheet" href="styles.css">

What Can CSS Do?

With CSS, you can:

  • Change colors, fonts, and text styles.
  • Create layouts, from simple grids to complex responsive designs.
  • Add transitions, animations, and effects to elements.
  • Control spacing, alignment, and positioning.

The possibilities with CSS are virtually limitless. It transforms plain HTML into beautiful, interactive websites.
