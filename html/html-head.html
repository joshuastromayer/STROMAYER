HTML Head

The <head> element in HTML is a container for metadata (data about data) and links to external resources. While the <head> element doesnâ€™t render anything on the webpage itself, it is critical for defining how the webpage behaves and appears.

What is the Head Section?

The <head> section is placed at the top of an HTML document, before the <body> section. It contains information about the document, such as its title, character encoding, linked stylesheets, scripts, and more.

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Webpage</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>

Key Elements of the Head Section

  • The Title: Defines the title of the document, which appears on the browser tab.
    <title>My Website</title>
  • Meta Tags: Provide metadata about the document, such as character encoding, description, or viewport settings.
    <meta charset="UTF-8">
    <meta name="description" content="Learn HTML tutorials">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Linking Stylesheets: Connect an external CSS file to style your webpage.
    <link rel="stylesheet" href="styles.css">
  • Adding Scripts: Link external JavaScript files or add inline JavaScript.
    <script src="script.js"></script>
  • Favicon: Add a small icon that appears in the browser tab.
    <link rel="icon" href="favicon.ico" type="image/x-icon">
  • SEO Metadata: Provide keywords and descriptions to help search engines understand your page.
    <meta name="keywords" content="HTML, CSS, Web Development">
    <meta name="author" content="Your Name">
  • Other Resources: Link fonts, icons, or other external resources.
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

Importance of the Head Section

The <head> section provides critical information that helps browsers, search engines, and other technologies understand your webpage. It also allows you to link external files and define global settings for your page.

Example: A Fully Configured Head Section

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Webpage</title>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Learn HTML tutorials">
    <link rel="icon" href="favicon.ico" type="image/x-icon">
    <link rel="stylesheet" href="styles.css">
    <script src="script.js"></script>
  </head>
  <body>
    <p>Welcome to my webpage!</p>
  </body>
</html>
