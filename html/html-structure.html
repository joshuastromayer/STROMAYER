Document Structure

<!DOCTYPE html>
<html>
<head>
  <title>Learning HTML</title>
</head>
<body>
  <h1>This is the Main Heading</h1>
  <div>
    <h3>This is a Sub-Heading</h3>
    <p>This is a paragraph.</p>
  </div>
</body>
</html>

Code Explained

This HTML code represents the foundation of a webpage. HTML, or HyperText Markup Language, is used to structure content on the web. Each line of this code serves a specific purpose, and every tag defines how the browser interprets and displays the content.

<!DOCTYPE html>

The <!DOCTYPE html> declaration informs the browser that this document is written in HTML5, the latest version of HTML. It sets the standard for how the browser should process and render the code.

<html>

The <html> tag serves as the root element of the entire document. All other elements are nested within this tag. At the end of the code, the </html> tag closes the document. This ensures the content is properly encapsulated.

<head>

The <head> section contains metadata and information about the webpage that is not directly displayed to the user. For instance, the <title> tag within the head defines the title of the webpage, which is shown on the browser tab.

<body>

The <body> section contains the main content of the webpage. Everything within this tag is displayed to the user in the browser.

Inside the <body>:

  • <h1>: This tag defines the main heading of the page. In this example, "This is the Main Heading" will appear as a large, bold title.
  • <div>: The <div> tag is a container used to group elements together. It helps organize the content and apply styles or layouts collectively.
  • <h3>: This tag is used for subheadings. It is smaller than <h1> but still used to structure content logically.
  • <p>: This tag defines a paragraph of text. In this example, it contains the text "This is a paragraph.", which will appear as a block of text on the webpage.

Closing Tags

For every opening tag (e.g., <h1>), there is a corresponding closing tag (e.g., </h1>). These closing tags ensure that the structure of the document remains valid and well-organized.

Bringing It All Together

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Learning HTML</title>
  </head>
  <body>
    <h1>This is the Main Heading</h1>
    <h2>This is a Sub-Heading</h2>
    <p>This is a paragraph.</p>
  </body>
</html>

When this code is processed by a browser, it will:

  • Display the title "Learning HTML" on the browser tab.
  • Show a large main heading: "This is the Main Heading".
  • Include a subheading: "This is a Sub-Heading", within a grouped container.
  • Render a paragraph of text: "This is a paragraph."

Save your code in a file named my-first-page.html and open it in your browser to see the results!
