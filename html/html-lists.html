HTML Lists

Lists are a fundamental feature of HTML, allowing you to organize and display content in a structured and readable format. HTML supports three types of lists:

  • Unordered Lists: Items are displayed with bullet points.
  • Ordered Lists: Items are displayed with numbers or letters, indicating their order.
  • Definition Lists: Used to display terms and their descriptions.

Unordered Lists

Unordered lists are created using the <ul> tag. Each item in the list is wrapped in an <li> (list item) tag.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

This code creates a list like this:

  • HTML
  • CSS
  • JavaScript

Ordered Lists

Ordered lists are created using the <ol> tag. Items are numbered automatically.

<ol>
  <li>Step 1: Open your editor.</li>
  <li>Step 2: Write some code.</li>
  <li>Step 3: Save your file.</li>
</ol>

This code produces the following list:

  1. Step 1: Open your editor.
  2. Step 2: Write some code.
  3. Step 3: Save your file.

Definition Lists

Definition lists consist of terms and their definitions. They use the <dl> (definition list), <dt> (definition term), and <dd> (definition description) tags.

<dl>
  <dt>HTML</dt>
  <dd>The standard markup language for web pages.</dd>
  <dt>CSS</dt>
  <dd>Styles the content of web pages.</dd>
  <dt>JavaScript</dt>
  <dd>Adds interactivity to web pages.</dd>
</dl>

This code creates a list like this:

HTML
The standard markup language for web pages.
CSS
Styles the content of web pages.
JavaScript
Adds interactivity to web pages.

When to Use Each List Type

  • Unordered Lists: Use when the order of items doesn’t matter.
  • Ordered Lists: Use when the order or sequence of items is important.
  • Definition Lists: Use for glossaries, descriptions, or term definitions.

Nested Lists

Sometimes, you might need to create a list inside another list. This is called a nested list. You can nest both ordered (<ol>) and unordered (<ul>) lists.

Example: A Nested Unordered List

<ul>
  <li>Fruits
    <ul>
      <li>Apples</li>
      <li>Bananas</li>
      <li>Oranges</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrots</li>
      <li>Broccoli</li>
    </ul>
  </li>
</ul>

Example: A Nested Ordered List

<ol>
  <li>Morning Routine
    <ol>
      <li>Wake up</li>
      <li>Brush teeth</li>
      <li>Exercise</li>
    </ol>
  </li>
  <li>Evening Routine
    <ol>
      <li>Dinner</li>
      <li>Read a book</li>
      <li>Go to bed</li>
    </ol>
  </li>
</ol>

Output

For the unordered list:

  • Fruits
    • Apples
    • Bananas
    • Oranges
  • Vegetables
    • Carrots
    • Broccoli

For the ordered list:

  1. Morning Routine
    1. Wake up
    2. Brush teeth
    3. Exercise
  2. Evening Routine
    1. Dinner
    2. Read a book
    3. Go to bed

Styling Lists

Lists can be styled with CSS. For example, you can change bullet points to other symbols, adjust spacing, or even remove the default list markers entirely.

<ul style="list-style-type: square;">
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

The above example changes the bullets to squares instead of circles. But we will go further in detail about styling in our CSS tutorials.

HTML Lists: Exercises

1. Create an Unordered List

Create an unordered list to display the following items: "Apples", "Bananas", "Cherries".

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

2. Create an Ordered List

Write HTML to create an ordered list showing the steps of making tea:

  • Boil water
  • Add a tea bag to a cup
  • Pour the boiled water into the cup
  • Wait 3–5 minutes
  • Remove the tea bag and enjoy

<ol>
  <li>Boil water</li>
  <li>Add a tea bag to a cup</li>
  <li>Pour the boiled water into the cup</li>
  <li>Wait 3–5 minutes</li>
  <li>Remove the tea bag and enjoy</li>
</ol>

3. Create a Definition List

Write a definition list for the following terms:

  • HTML: The standard markup language for web pages.
  • CSS: Styles the content of web pages.
  • JavaScript: Adds interactivity to web pages.

<dl>
  <dt>HTML</dt>
  <dd>The standard markup language for web pages.</dd>
  <dt>CSS</dt>
  <dd>Styles the content of web pages.</dd>
  <dt>JavaScript</dt>
  <dd>Adds interactivity to web pages.</dd>
</dl>

4. Style a List

Create an unordered list with the items "Red", "Green", and "Blue". Use inline CSS to change the bullet points to squares.

<ul style="list-style-type: square;">
  <li>Red</li>
  <li>Green</li>
  <li>Blue</li>
</ul>

5. Combine List Types

Create a list structure where the main list is ordered, and one of the items contains an unordered sublist. For example:

  1. Plan a meal
  2. Buy groceries:
    • Vegetables
    • Fruits
    • Spices
  3. Cook and enjoy

<ol>
  <li>Plan a meal</li>
  <li>Buy groceries:</li>
    <ul>
      <li>Vegetables</li>
      <li>Fruits</li>
      <li>Spices</li>
    </ul>
  <li>Cook and enjoy</li>
</ol>
