HTML IDs

The id attribute is used to uniquely identify an element on a web page. Unlike classes, which can be shared by multiple elements, an id must be unique within the document.

What is an ID?

An id is an attribute that provides a unique identifier for an HTML element. This identifier can be used for styling, scripting, or linking to specific parts of a page. IDs are especially useful for JavaScript operations or when creating anchor links.

<div id="unique-id">
  <p>This div has a unique ID.</p>
</div>

Using IDs for Styling

You can target an element with a specific id in CSS using the # selector. For example:

<style>
  #unique-id { color: red; }
</style>

<div id="unique-id">This text is red.</div>

Using IDs for JavaScript

IDs are commonly used to target elements in JavaScript. For example, you can change the text of a specific element using its ID:

<div id="greeting">Hello!</div>
<script>
  document.getElementById("greeting").innerText = "Welcome!";
</script>

Important Note on Styling with IDs and Classes

When using CSS to style elements:

  • To target an element with a specific id, use the # symbol followed by the ID name.
  • To target elements with a specific class, use the . (dot) symbol followed by the class name.
/* Targeting an ID */
#unique-id {
  color: blue;
}

/* Targeting a class */
.shared-class {
  font-size: 16px;
}

Using IDs for Anchor Links

IDs can also be used to create anchor links that jump to specific sections of a page. For example:

<a href="#section1">Go to Section 1</a>
<h2 id="section1">Section 1</h2>

Best Practices for Using IDs

While IDs are powerful, they should be used carefully to ensure clean and maintainable code:

  • Use IDs for unique elements that need to be targeted specifically.
  • Avoid using IDs for styling multiple elements; use classes for that purpose.
  • Ensure that each ID is unique within the document to prevent conflicts.
  • Use meaningful names for IDs to make your code easier to understand.

Example: Combining IDs and Classes

You can use both classes and IDs on the same element to combine their benefits. For example:

<style>
  #special-text { color: blue; font-weight: bold; }
  .highlight { background-color: yellow; }
</style>

<p id="special-text" class="highlight">This text is blue, bold, and highlighted.</p>

HTML IDs: Exercises

1. Create a Unique ID

Create a <div> with the ID "welcome-message" that contains the text "Welcome to My Website!".

<div id="welcome-message">Welcome to My Website!</div>

2. Use an ID for Styling

Create a <p> element with the ID "highlighted-text" that says "This is important text.". Try use CSS to make the text bold and red.

<style>
  #highlighted-text { color: red; font-weight: bold; }
</style>

<p id="highlighted-text">This is important text.</p>

3. Use an ID in JavaScript

Create a <button> with the ID "change-text-button". Add a <div> with the ID "target-div" that says "Original Text". Then take a look at the JavaScript below that changes the text inside the <div> to "Updated Text" when the button is clicked.

<div id="target-div">Original Text</div>
<button id="change-text-button">Change Text</button>
<script>
  document.getElementById("change-text-button").addEventListener("click", function() {
    document.getElementById("target-div").innerText = "Updated Text";
  });
</script>

4. Create an Anchor Link

Create a link that jumps to a section on the same page. The link should say "Go to About Section". Create a section with the ID "about-section" that contains the text "About Us".

<a href="#about-section">Go to About Section</a>
<div id="about-section">About Us</div>

5. Combine IDs and Classes

Create a <p> element with the ID "unique-paragraph" and the class "highlight". Make the text say "This is a unique highlighted paragraph.". Try use CSS to style it with a yellow background (class) and blue text (ID).

<style>
  #unique-paragraph { color: blue; }
  .highlight { background-color: yellow; }
</style>

<p id="unique-paragraph" class="highlight">This is a unique highlighted paragraph.</p>
