HTML Elements

HTML is built around the concept of elements. An HTML element consists of a start tag, some content, and an end tag. Together, they form the structure and content of a webpage. Understanding elements is the foundation of learning HTML.

What Is an HTML Element?

An HTML element is a combination of a start tag, content, and an end tag. For example:

<h1>This is a heading</h1>

Here:

  • Start tag: <h1>
  • Content: "This is a heading"
  • End tag: </h1>

The start tag defines the beginning of the element, and the end tag defines its closure. Some elements, such as <br>, are self-closing and don’t require an end tag.

Nesting Elements

HTML elements can be nested within each other to create a structure. For example:

<div>
  <h1>Welcome</h1>
  <p>This is a paragraph inside a div.</p>
</div>

In this example, the <h1> and <p> elements are nested inside the <div> element.

Attributes in Elements

Elements can have attributes that provide additional information. Attributes are included in the opening tag. For example:

<a href="https://example.com">Visit Example</a>

Here, href is an attribute that specifies the link's destination. Attributes always follow this pattern: name="value".

Empty Elements

Some elements don’t have any content or closing tags. They are called empty elements. Examples include:

<br> <img src="image.png" alt="Description">

Empty elements are useful for things like line breaks (<br>) or adding images (<img>).

Common Mistakes

  • Forgetting to close tags: Always close elements unless they are self-closing.
  • Incorrect nesting: Make sure nested elements are properly structured. For example, this is invalid: <p><div>Text</div></p>.
  • Misspelling tag names: Tags are case-insensitive but should be consistent (prefer lowercase).
